# 🔍 查重服务概述

## 服务定位

查重服务是形式审查的子服务之一，负责对上传的文档进行文本相似度检测，识别重复申报、抄袭等情况。

## 功能特性

### 1. 多格式文本提取

- **PDF 解析**: 复用 `common/file_handler/pdf_parser.py`
- **DOCX 解析**: 复用 `common/file_handler/docx_parser.py`
- **纯文本提取**: 自动清理格式，保留核心内容

### 2. Section 区域提取

- **标题匹配**: 使用正则表达式匹配文档章节标题
- **可配置**: 支持自定义待查重区域
- **精确提取**: 只提取有价值的查重内容

### 3. 句子级比对

- **精确匹配**: 基于句子级别的文本比对
- **位置追溯**: 记录重复内容的来源位置
- **阈值可配**: 支持自定义高/中相似度阈值

### 4. 批量查重

- **全量比对**: 所有文档两两比对
- **结果分类**: 高相似度 / 中相似度 / 低相似度
- **性能优化**: 高效文本比对算法


## Section 配置

系统支持通过配置文件定义待查重的文档区域。

### 默认配置

适用于项目申报书类文档，需要查重的区域如下：

| Section | 起始标题 | 结束标题 |
|---------|----------|----------|
| 项目立项背景及意义 | 项目立项背景及意义 | 项目简介 |
| 一、项目实施内容及目标 | 一、项目实施内容及目标 | 二、申报单位及合作单位基础 |
| 二、申报单位及合作单位基础 | 二、申报单位及合作单位基础 | 三、项目实施计划及保障措施和风险分析 |
| 三、项目实施计划及保障措施和风险分析 | 三、项目实施计划及保障措施和风险分析 | 文档结尾 |

### 配置定义

配置文件位于 `src/services/plagiarism/config.py`：

```python
PLAGIARISM_SECTION_CONFIG: Dict[str, Dict] = {
    "default": {
        "name": "默认配置",
        "sections": [
            {
                "name": "项目立项背景及意义",
                "start_pattern": r"项目立项背景及意义",
                "end_pattern": r"项目简介",
            },
            # ... 更多 section
        ]
    },
}
```

### 自定义配置

支持通过 API 参数传入自定义 section 配置（详见 API 文档）。
