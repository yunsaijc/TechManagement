# 🔍 查重服务概述

## 服务定位

查重服务是形式审查的子服务之一，负责对上传的文档进行文本相似度检测，识别重复申报、抄袭等情况。

## 功能特性

### 1. 多格式文本提取

- **PDF 解析**: 复用 `common/file_handler/pdf_parser.py`
- **DOCX 解析**: 复用 `common/file_handler/docx_parser.py`
- **纯文本提取**: 自动清理格式，保留核心内容

### 2. 向量化比对

- **向量化模型**: 复用 `common/llm/embedding_factory.py`
- **相似度计算**: 余弦相似度
- **阈值可配**: 支持自定义高/中相似度阈值

### 3. 批量查重

- **全量比对**: 所有文档两两比对
- **结果分类**: 高相似度 / 中相似度 / 低相似度
- **性能优化**: 向量化批量处理

## 业务流程

```
┌─────────────────────────────────────────────────────────────────┐
│                         查重流程                                 │
└─────────────────────────────────────────────────────────────────┘

     用户上传文件（多个）
           │
           ▼
    ┌─────────────┐
    │  文件接收   │  ← 验证格式
    └──────┬──────┘
           │
           ▼
    ┌─────────────┐
    │  文本提取   │  ← PDF/DOCX 解析
    └──────┬──────┘
           │
           ▼
    ┌─────────────┐
    │  向量化处理 │  ← Embedding
    └──────┬──────┘
           │
           ▼
    ┌─────────────┐
    │  相似度计算 │  ← 两两比对
    └──────┬──────┘
           │
           ▼
    ┌─────────────────────────────┐
    │        结果聚合             │
    │  - 高相似度对列表          │
    │  - 中相似度对列表          │
    └──────────────┬────────────┘
                   │
                   ▼
           查重结果输出
```
