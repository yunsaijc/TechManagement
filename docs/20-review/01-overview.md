# 🔍 形式审查服务概述

## 服务定位

形式审查服务是科技管理系统的核心服务之一，负责对上传的文档进行自动化的形式审查，包括签字、盖章检查，前置条件验证，一致性核对等。

## 功能特性

### 1. 多模态智能审查

- **签字检测**: 自动识别文档中的签名区域，验证签字是否存在
- **盖章检测**: 识别印章、戳记，验证盖章是否符合要求
- **版式分析**: 分析文档版式，处理复杂排版

### 2. 规则 + AI 混合

- **规则引擎**: 基于预定义规则的精确检查
- **LLM 补充**: 多模态大模型处理复杂版式和语义理解
- **协同决策**: Agent 协调规则和 LLM，取长补短

### 3. 文档类型支持

| 文档类型 | 检查项 |
|----------|--------|
| 专利证书 | 签字、盖章、专利号一致性 |
| 验收报告 | 前置条件、签字、盖章 |
| 行政许可 | 许可文件完整性 |
| 检索报告 | 检索结果完整性 |
| 奖励证书 | 证书一致性 |
| 合同协议 | 签字、盖章、条款完整性 |

## 业务流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        形式审查流程                              │
└─────────────────────────────────────────────────────────────────┘

     用户上传文件
           │
           ▼
    ┌─────────────┐
    │  文件接收   │  ← 验证格式、大小
    └──────┬──────┘
           │
           ▼
    ┌─────────────┐
    │  文档分类   │  ← 确定文档类型
    └──────┬──────┘
           │
           ▼
    ┌─────────────┐
    │  版式解析   │  ← 提取文字、图像区域
    └──────┬──────┘
           │
           ▼
    ┌─────────────────────────────────────┐
    │          Agent 决策                  │
    │  ┌─────────┐    ┌─────────────┐   │
    │  │ 规则检查 │ +  │   LLM 分析   │   │
    │  └────┬────┘    └──────┬──────┘   │
    │       └─────────────────┘          │
    └──────────────┬────────────────────┘
                   │
                   ▼
    ┌─────────────────────────────┐
    │        结果聚合             │
    │  - 检查项列表              │
    │  - 总结和建议             │
    └──────────────┬────────────┘
                   │
                   ▼
           审查结果输出
```

## 模块结构

```
src/
├── common/                         # 通用层
│   ├── vision/                    # 底层视觉能力
│   │   ├── detector.py            # YOLO 检测器
│   │   └── multimodal.py          # 多模态 LLM
│   └── detectors/                 # 通用检测器（Layer 4）
│       ├── signature.py            # 签字检测器
│       └── stamp.py               # 盖章检测器
│
└── services/review/               # 形式审查服务
    ├── agent.py                   # 审查 Agent (Layer 6)
    ├── service.py                 # 服务入口
    ├── extractor.py               # 内容提取（OCR + 文字解析）
    ├── parser.py                  # 文档解析
    └── rules/                     # 规则引擎 (Layer 5)
        ├── base.py                # 规则基类
        ├── registry.py            # 规则注册表
        └── checkers/              # 具体检查器
            ├── signature.py       # 签字检查
            ├── stamp.py           # 盖章检查
            ├── prerequisite.py    # 前置条件
            └── consistency.py     # 一致性检查
```

## 核心技术

| 技术 | 用途 |
|------|------|
| **LangChain LCEL** | Agent 编排、链式调用 |
| **多模态 LLM** | 复杂版式分析、语义理解 |
| **YOLO/GroundingDino** | 目标检测（签名、印章） |
| **PaddleOCR** | 文字识别 |
| **PyMuPDF** | PDF 解析 |

## API 接口

```
POST /api/v1/review
Content-Type: multipart/form-data

参数:
  - file: 文件 (必填)
  - document_type: 文档类型 (可选，自动识别)
  - check_items: 检查项列表 (可选，默认全部)

响应:
{
  "id": "review_xxx",
  "document_type": "patent_certificate",
  "results": [
    {
      "item": "signature",
      "status": "passed",
      "message": "检测到签名",
      "evidence": {"regions": [...]}
    }
  ],
  "summary": "审查完成，发现 X 个问题",
  "suggestions": [...]
}
```
