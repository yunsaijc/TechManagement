# 🔍 形式审查服务概述

## 服务定位

形式审查服务是科技管理系统的核心服务之一，负责项目申报材料的自动化审查。当前建议将其拆分为两层能力：

- **附件级形式审查**：对单个附件进行签字、盖章、完整性等检查
- **项目级形式审查**：按项目类型组织材料、汇总附件结果，并执行项目级规则

查重服务仍作为形式审查体系下的独立子服务存在。

## 子服务

### 1. 附件级形式审查 (Document Review)

对单个附件进行形式检查，包括签字、盖章、完整性、一致性等。

| 文档类型 | 检查项 |
|----------|--------|
| 专利证书 | 签字、盖章、专利号一致性 |
| 验收报告 | 前置条件、签字、盖章 |
| 行政许可 | 许可文件完整性 |
| 检索报告 | 检索结果完整性 |
| 奖励证书 | 证书一致性 |
| 合同协议 | 签字、盖章、条款完整性 |

详细文档：[附件级形式审查 →](format/01-overview.md)

### 2. 项目级形式审查 (Project Review)

对整个项目进行审查，按项目类型判断材料是否齐套、条件是否满足，并汇总附件级检查结果。

- **输入**: 项目信息、附件清单、外部校验结果
- **核心能力**: 材料齐套性、条件性附件、项目级规则、跨附件汇总

详细文档：[项目级形式审查 →](project-review/01-overview.md)

### 3. 查重 (Plagiarism)

对多个文档进行文本相似度检测，识别重复申报、抄袭等情况。

- **支持格式**: PDF、DOCX
- **比对方式**: 向量化 + 余弦相似度
- **场景**: 跨项目查重、疑似抄袭检测

详细文档：[查重服务 →](plagiarism/01-overview.md)

## 业务流程

```
┌─────────────────────────────────────────────────────────────────┐
│                       形式审查流程                                │
└─────────────────────────────────────────────────────────────────┘

     用户提交项目 / 附件
           │
           ▼
    ┌─────────────────────┐
    │   项目级形式审查     │
    │  - 材料齐套检查      │
    │  - 项目级规则        │
    │  - 附件结果汇总      │
    └──────────┬──────────┘
               │
      ┌────────▼────────┐
      │  附件级形式审查   │
      │ - 签字检查       │
      │ - 盖章检查       │
      │ - 完整性检查     │
      └────────┬────────┘
               │
      ┌────────▼────────┐
      │      查重        │
      │   (按需调用)     │
      └────────┬────────┘
               │
               ▼
           审查结果输出
```

```
│   ├── file_handler/              # 文件处理

│   │   └── docx_parser.py        # DOCX 解析
│   ├── llm/                      # LLM 能力
│   │   └── embedding_factory.py   # 向量化
│   ├── vision/                   # 视觉能力
│   │   ├── detector.py          # YOLO 检测器
│   │   └── multimodal.py        # 多模态 LLM
│   └── extractors/               # 统一提取器
│       ├── signature.py          # 签字提取器
│       ├── stamp.py              # 盖章提取器
│       └── field.py              # 字段提取器
│
└── services/
    ├── review/                   # 形式审查服务
    │   ├── agent.py             # 附件级 ReviewAgent
    │   ├── rules/               # 规则引擎
    │   │   ├── base.py
    │   │   ├── registry.py
    │   │   └── checkers/
    │   │       ├── signature.py

    │   │       └── work_unit.py
    │   └── api.py

    └── plagiarism/               # 查重服务
        ├── agent.py             # PlagiarismAgent
        ├── comparator.py        # 比对逻辑
        └── api.py
```

## 核心技术

|| 技术 | 用途 |
|------|------|------|
| **LangChain LCEL** | Agent 编排、链式调用 |
| **多模态 LLM** | 复杂版式分析、语义理解 |
| **YOLO/GroundingDino** | 目标检测（签名、印章） |
| **PaddleOCR** | 文字识别 |
| **PyMuPDF** | PDF 解析 |
| **Embedding** | 文本向量化 |
