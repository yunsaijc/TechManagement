# 🔍 形式审查服务概述

## 服务定位

形式审查服务是科技管理系统的核心服务之一，负责对上传的文档进行自动化的形式审查，包括**格式审查**和**查重**两大子服务。

## 子服务

### 1. 格式审查 (Format Review)

对单个文档进行格式检查，包括签字、盖章、一致性等。

| 文档类型 | 检查项 |
|----------|--------|
| 专利证书 | 签字、盖章、专利号一致性 |
| 验收报告 | 前置条件、签字、盖章 |
| 行政许可 | 许可文件完整性 |
| 检索报告 | 检索结果完整性 |
| 奖励证书 | 证书一致性 |
| 合同协议 | 签字、盖章、条款完整性 |

详细文档：[格式审查 →](format/01-overview.md)

### 2. 查重 (Plagiarism)

对多个文档进行文本相似度检测，识别重复申报、抄袭等情况。

- **支持格式**: PDF、DOCX
- **比对方式**: 向量化 + 余弦相似度
- **场景**: 跨项目查重、疑似抄袭检测

详细文档：[查重服务 →](plagiarism/01-overview.md)

```
    ┌─────────────────────┐         ┌─────────────────────┐

               │                               │
               └───────────────┬───────────────┘
                   ▼
           审查结果输出
```

## 核心技术

|| 技术 | 用途 |
|------|------|------|
| **LangChain LCEL** | Agent 编排、链式调用 |
| **多模态 LLM** | 复杂版式分析、语义理解 |
| **YOLO/GroundingDino** | 目标检测（签名、印章） |
| **PaddleOCR** | 文字识别 |
| **PyMuPDF** | PDF 解析 |
| **Embedding** | 文本向量化 |
