# 沙盘推演总览

## 一、目标

沙盘推演用于回答“如果实施某项政策动作，未来会怎样变化”。

它面向的是干预性、反事实和因果型分析，而不是单纯的趋势展示。

## 二、核心问题

1. 如果调整指南方向或资助配额，会影响哪些主题
2. 如果改变评审权重或准入约束，会如何影响申报与立项结构
3. 如果对重点方向补人才、补协作、补资源，转化是否会改善
4. 不同政策组合的效果差异和副作用是什么

## 三、标准输出

- `baseline`
- `intervention`
- `counterfactual`
- `impact_assessment`
- `causal_path`
- `side_effects`

## 四、能力模块

建议拆为以下能力模块：

1. 政策动作建模
2. 场景定义与约束管理
3. 因果假设与传导结构
4. 干预求解与反事实生成
5. 多方案比较与稳健性分析

## 五、与趋势预判的关系

沙盘推演依赖趋势预判提供基线世界；
没有可信的基线，就无法定义可信的反事实。
