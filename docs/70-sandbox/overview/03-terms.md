# 70 核心术语

## 一、世界与时间

- `Baseline World`
  在当前政策 regime 不变时，由真实数据支撑的基线世界。
- `Counterfactual World`
  在 baseline 上施加政策动作后形成的反事实世界。
- `Time Window`
  观测、比较或推演的时间窗。
- `Application Cohort`
  同一申报年份的一批项目，用于做 cohort 口径结果观察。
- `Event Year`
  某业务事件真实发生的年份，如评审年、合同年。

## 二、对象与结构

- `Topic`
  政策/指南方向，不等于任意自由文本标签。
- `Project`
  从申报到后续结果的生命周期主实体。
- `Institution`
  承担单位或合作单位。
- `Person`
  负责人或参与人员。
- `Topic Panel`
  基于 `topic × year` 的统计面板。
- `Portfolio`
  全省科研投资组合，领导真正决策的对象。

## 三、分析与推演

- `Trend`
  对 baseline world 的未来演化判断。
- `Signal`
  结构变化、异常变化或风险累积的可解释信号。
- `Risk Alert`
  基于真实观测或代理结构形成的预警。
- `Simulation`
  在 baseline 上施加结构化政策动作后的反事实比较。
- `Policy Action`
  单个政策动作的结构化表达。
- `Policy Package`
  一次情景推演中组合出现的政策动作集合。
- `Scenario Contract`
  一次完整推演的输入合同，至少包含意图、范围、动作、约束、评估与验证声明。
- `Constraint`
  预算、配额、风险、合规等限制。
- `Crowding-out`
  固定预算或容量约束下，某些主题受益以其他主题受损为代价的挤出效应。

## 四、证据与表达

- `Observed`
  可由真实业务表直接观测到的指标或事实。
- `Proxy`
  只能通过评分、图结构或派生计算近似反映的代理量。
- `Structural Assumption`
  依赖结构机制或规则假设而非直接历史识别的推断。
- `Unsupported`
  当前数据无法支撑的指标或结论。
- `Evidence Chain`
  从项目事实、聚合口径、规则链路到结论表达的完整可追溯链。

## 五、交付层

- `Leadership Briefing`
  面向领导的结构化简报页面。
- `Copilot QA`
  基于系统结果和证据链进行追问与展开说明的问答能力。

`Leadership Briefing` 与 `Copilot QA` 都不是分析本体，只能消费分析与推演结果。
