# 📋 智能分组与专家匹配服务概述

## 服务定位

智能分组与专家匹配服务是科技管理系统的核心服务之一，负责对申报项目进行智能分组，并为每个分组匹配最合适的评审专家。

### 核心能力

1. **智能学科分组**：分析项目内容，实现组内数量与质量双均衡
2. **全局最优专家匹配**：以评审组为单元，确保每组有足够的懂行专家
3. **智能关系回避**：自动检测并回避师生、历史合作等关系

### 解决的核心问题

| 现状痛点 | 解决方案 |
|----------|----------|
| 按三级学科分组，同组>30项时无法处理 | AI分析项目内容，智能分组 |
| "神仙打架"或专家不懂行 | 全局最优专家匹配 |
| 师生关系、利益冲突 | 智能回避关系检测 |

---

## 功能特性

### 1. 智能分组 (Grouping)

- **项目内容分析**：使用 LLM 提取项目核心创新点、技术方向、研究领域
- **自动分组计算**：根据项目数量自动计算最优分组数
- **质量均衡分配**：各组综合质量得分尽量均衡
- **学科内聚性**：同组分尽量主题相关

### 2. 专家匹配 (Matching)

- **专家画像构建**：从研究领域、论文、擅长专业构建专家向量
- **全局最优匹配**：以组为单元的整体优化匹配
- **关系回避**：支持师生、历史合作、单位回避
- **可解释性**：匹配结果附带原因说明

## 技术选型

| 技术 | 用途 |
|------|------|
| **LangChain LCEL** | Agent 编排、链式调用 |
| **多模态 LLM** | 项目内容分析、专家画像构建 |
| **BGE-M3 / OpenAI** | 文本向量化 |
| **sklearn** | K-means 聚类、层次聚类 |
| **scipy** | 约束优化、线性规划 |
| **FAISS** | 向量相似度检索 |
