# 70 核心术语

## 一、通用术语

- `TimeWindow`：时间窗，用于定义观测期、比较期和预测期
- `Baseline`：不施加政策干预时的基线世界
- `Signal`：结构变化、异常变化或风险累积的可解释信号
- `Forecast`：在基线条件下对未来窗口的预测结果
- `Intervention`：人为施加的政策动作
- `Counterfactual`：施加干预后的反事实结果

## 二、趋势预判术语

- `Topic`：主题、指南方向、学科方向或政策关注方向
- `Topic Migration`：主题间热度、资源、人才或项目重心的迁移
- `Lifecycle`：主题从萌芽、升温、成熟到衰减的演化阶段
- `Risk Alert`：对未来窗口有解释依据的前瞻风险提示

## 三、沙盘推演术语

- `PolicyInstrument`：政策动作的抽象表达
- `Scenario`：一组政策动作与约束组合形成的推演场景
- `Constraint`：预算、容量、周期、资格等约束条件
- `Impact Assessment`：干预后相对基线的变化评估
- `Causal Path`：政策动作影响结果变量的传导路径

## 四、交付层术语

- `Briefing`：供领导直接消费的结构化简报
- `Copilot QA`：基于分析结果和证据进行追问的问答能力

`Briefing` 与 `Copilot QA` 均不属于分析本体，只能消费分析层结果。
