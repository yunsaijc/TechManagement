# 趋势预判总览

## 一、目标

趋势预判用于回答“如果不做政策干预，系统接下来会怎样演化”。

它面向的是描述性与预测性分析，而不是政策干预仿真。

## 二、核心问题

1. 热点正在从哪些主题迁移到哪些主题
2. 哪些方向在升温，哪些方向在衰减
3. 人才结构、协作结构和转化结构是否正在恶化
4. 哪些风险信号会在未来窗口中放大

## 三、标准输出

- `state_snapshot`
- `trend_signal`
- `forecast`
- `risk_alert`
- `explanation`

## 四、能力模块

建议拆为以下能力模块：

1. 结构变化检测
2. 热点迁移分析
3. 主题生命周期识别
4. 人才/转化风险前瞻
5. 未来窗口预测

## 五、与其他模块的关系

- 趋势预判为沙盘推演提供基线世界
- 趋势预判结果应先于简报和问答
- 趋势预判不是规则告警页面，也不是报告模板本身
