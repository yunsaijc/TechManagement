# 📊 数据接入概述

## 数据来源

本系统数据来源于甲方（河北省科技厅）奖励评审系统数据库，用于：
- **测试阶段**：验证功能正确性
- **训练阶段**：优化 AI 模型效果
- **交付阶段**：提供 API 服务（不含原始数据）

## 接入原则

| 原则 | 说明 |
|------|------|
| **只读** | 禁止任何写入操作，保护甲方数据 |
| **按需加载** | 根据服务需求加载对应数据，不一次性接入全部 |
| **只读** | 开发调试直连甲方服务器，无本地副本 |
| **可扩展** | 支持后续接入更多数据库（如项目评审） |
